Detailed Description

Plugin for clutter management using hit accumulations: HAC This plugin is intended for real-time incrementation of the HAC files.

Filtering is better done through the use of a quality plugin. Register in the RAVE PGF with: % pgf_registry -a -H http://<host>:<port>/RAVE –name=eu.opera.odc.hacincrementor -m odc_hac_pgf_plugin -f generate -d 'Hit-accumulation incrementor'
