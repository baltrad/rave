Detailed Description

Mapping between a proj.4 definition id and a wkt id.

For example defid = +lat_0 and wktid = latitude_of_projection_origin. The definition can be found in Table F.1. Grid Mapping Attributes in CF conventions 1.7: http://cfconventions.org/Data/cf-conventions/cf-conventions-1.7/build/apf.html
