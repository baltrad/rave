Detailed Description

Plugin for applying quality control on a volume that is initiated from the beast framework.

Register in the RAVE PGF with: % pgf_registry -a -H http://<host>:<port>/RAVE –name=eu.baltrad.beast.applyqc_plugin –strings=source,date,time,anomaly-qc -m rave_pgf_apply_qc -f generate -d 'Apply quality controls on a polar volume'
