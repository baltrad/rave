Detailed Description

A composite quality plugin for generating the PIA quality field.

This is no interactive registry, instead you will have to modify the xml file in COMPOSITE_QUALITY_REGISTRY manually.

<?xml version='1.0' encoding='UTF-8'?> <rave-pgf-composite-quality-registry> <quality-plugin name="ropo" class="ropo_pgf_composite_quality_plugin" /> <quality-plugin name="pia" class="rave_pia_quality_plugin" /> </rave-pgf-composite-quality-registry>
