Detailed Description

A registry for keeping track on areas that are divided into a number of tiles for faster processing.

This is no interactive registry, instead you will have to modify the xml file'manually.

<?xml version='1.0' encoding='UTF-8'?> <rave-tile-registry>

<tile id="1" extent="971337.728807,7196461.17902,1993337.7288084999,9112461.1790100001" > <tile id="2" extent="1993337.7288084999,7196461.17902,3015337.72881,9112461.1790100001" > .... </area> </rave-tile-registry>
