Detailed Description

Plugin for generating a composite that is initiated from the beast framework.

Register in pgf with –name=eu.baltrad.beast.generatecomposite –strings=area,quantity,method,date,time,selection,anomaly-qc –floats=height -m rave_pgf_composite_plugin -f generate
